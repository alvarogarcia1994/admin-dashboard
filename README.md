# admin-dashboard
Admin dashboard project from The Odin Project.

# EN
- Project from final module of Intermediate HTML and CSS from The Odin Project. The main objective consists to design an admin dashboard putting into practice advanced layout and interactive design techniques.

# Features
**Advanced Layout:** Intensive use of **CSS Grid** for the overall structure and **Flexbox** for specific components.
**Interactivity:** Implementation of dynamic **hover effects** with smooth transitions on icons, sidebar options, and header buttons.
**Dynamic UI:** Includes a sidebar, search bar, notifications area, project cards, and a trending section.

# Built with
- HTML5
- CSS3 (Grid and Flexbox)

# ES
- Este proyecto es el módulo final del curso de HTML y CSS Intermedio de The Odin Project. El objetivo principal fue diseñar una interfaz de panel de administración (Admin Dashboard) desde cero, poniendo en práctica técnicas avanzadas de maquetación y diseño responsivo.

# Características
**Maquetación Avanzada**: Uso intensivo de **CSS Grid** para la estructura general y **Flexbox** para alinear componentes específicos.
**UI Dinámica**: Interfaz moderna que incluye una barra lateral (sidebar), barra de búsqueda, área de notificaciones, tarjetas de proyectos y sección de tendencias.
**Interactividad y Microinteracciones**: Implementación de efectos hover dinámicos con transiciones suaves en los iconos, las opciones de la barra lateral (sidebar) y los botones principales del encabezado (header).

# Tecnologías utilizadas
- HTML5
- CSS3 (Grid y Flexbox)
